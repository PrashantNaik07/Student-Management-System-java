# 🎓 Student Management System (Java)

## 📌 Project Overview
The Student Management System is a console-based Java application that helps manage student records efficiently. It performs basic CRUD (Create, Read, Update, Delete) operations and stores data permanently using file handling.

This project is developed as part of a Java internship to demonstrate core concepts like OOP, Collections, and File Handling.

---

## 🚀 Features

- Add new student
- View all students
- Search student by ID
- Update student details
- Delete student record
- Prevent duplicate student IDs
- Save data to file (persistent storage)
- Load data automatically on startup

---

## 🛠️ Technologies Used

- Java (JDK 8 or above)
- Object-Oriented Programming (OOP)
- ArrayList (Collections Framework)
- File Handling (Serialization)
- VS Code / Any Java IDE
